# Muneesh Portfolio

Clean, modern portfolio site built with Next.js, Tailwind, and Framer Motion.

## Quickstart

```bash
npm install
npm run dev
```

## Notes

- Update content in `app/page.tsx`.
- Replace `/public/ai-profile.png` and `/public/Resume.pdf` with your assets.
- Deploy anywhere that supports Next.js (Vercel recommended).
